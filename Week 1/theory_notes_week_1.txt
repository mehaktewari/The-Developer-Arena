ðŸ”¹ What is Python?
Python is a high-level, interpreted programming language known for its simplicity and readability. 
It is widely used in web development, data science, AI/ML, automation, and more.

Python executes code line by line (interpreted) and does not require manual compilation.

--------------------------------------

ðŸ”¹ Features of Python:
- Easy to learn and use
- Cross-platform (works on Windows, macOS, Linux)
- Open-source and free
- Huge library support (NumPy, Pandas, TensorFlow)
- Object-Oriented and Dynamically Typed
