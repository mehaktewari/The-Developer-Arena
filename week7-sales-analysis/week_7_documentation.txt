--------------------------------------------------
PROJECT DESCRIPTION
--------------------------------------------------
The Sales Data Analysis Dashboard is an advanced Python-based
data analytics project that loads, cleans, analyzes, and visualizes
sales data using pandas.

The project simulates real-world business scenarios, allowing analysis
of total sales, monthly trends, category-wise insights, top products,
and customer behavior.

It also demonstrates professional report generation and visualization
techniques for data-driven decision-making.

--------------------------------------------------
OBJECTIVES
--------------------------------------------------
• Learn practical data analysis using pandas
• Perform data cleaning and preprocessing (handle missing values, duplicates)
• Generate actionable business insights from raw data
• Create visual reports using matplotlib
• Export Excel and image reports for decision making

--------------------------------------------------
FEATURES
--------------------------------------------------
✓ Load sales data from CSV and Excel files
✓ Clean missing values and duplicate entries
✓ Calculate total, average, and monthly sales
✓ Analyze category-wise and top-product sales
✓ Track monthly trends and growth rates
✓ Generate visualizations: bar charts, line graphs, histograms
✓ Export professional Excel reports
✓ Maintain modular and scalable project structure

--------------------------------------------------
TOOLS & LIBRARIES USED
--------------------------------------------------
• pandas – data manipulation and analysis
• numpy – numerical operations
• matplotlib – data visualization
• openpyxl – Excel export support
• Jupyter Notebook – exploratory and advanced analysis
