# Contact Management System — Week 3 Internship Task

## Overview
This project is a **Contact Management System** developed as part of **Week 3 of the internship program**.  
It focuses on **functions, dictionaries, file handling, validation, and modular design** in Python.

---

## Objectives
- Use functions to organize reusable logic
- Store structured data using dictionaries
- Perform CRUD operations
- Implement input validation
- Persist data using JSON files

---

## Technologies Used
- Python 3
- JSON
- CSV
- Regular Expressions

## Features
- Add, search, update, delete contacts
- Partial name search
- Phone & email validation
- Save/load data automatically
- Export contacts to CSV
- Menu-driven interface
