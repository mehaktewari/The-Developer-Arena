==================================================
PROJECT DESCRIPTION
==================================================

The Weather Dashboard Application is a command-line based Python project
that fetches real-time weather data using an external weather API
(OpenWeatherMap or similar). The application displays current weather,
5-day forecasts, and detailed weather information in a clean and
user-friendly format.

This project focuses on integrating external libraries, working with APIs,
handling JSON data, managing environment variables, and implementing
robust error handling.

==================================================
MODULE DESCRIPTION
==================================================

1. weather_api.py
-----------------
• Handles all API communication
• Makes HTTP requests to weather API
• Implements caching mechanism
• Handles API and network errors

2. weather_parser.py
--------------------
• Parses raw JSON API responses
• Converts temperature units
• Formats date and time values
• Extracts relevant weather data

3. weather_display.py
---------------------
• Displays weather information in formatted output
• Uses icons and symbols for weather conditions
• Shows current weather and forecasts

4. config.py
------------
• Stores API-related configuration
• Loads environment variables

5. main.py
----------
• Entry point of the application
• Handles user interaction and menu system
• Integrates all modules

==================================================
HOW TO RUN THE PROJECT
==================================================

1. Get a free API key from OpenWeatherMap
2. Copy .env.example to .env
3. Add your API key inside .env file
4. Install required libraries:
   pip install -r requirements.txt
5. Run the application:
   python -m weather_app.main
